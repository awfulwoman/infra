# Radicale

[Radicale](https://radicale.org) is a lightweight, self-hosted CalDAV/CardDAV server. Used here as the backend for personal reminders (VTODO collections), replacing macOS EventKit/Apple Reminders — `gw reminders` talks to it via the `caldav` Python library, and the iOS client is [NowThis](https://nowthis.app/).

## Ports

Internal port `5232`. Exposed via Traefik at `radicale.<domain>`.

## Environment

Configured entirely via `RADICALE_CONFIG_*` environment variables (no mounted config file):

- Storage backend: filesystem, at `/data/collections`
- Auth: `htpasswd` with bcrypt, file at `/config/users`

## Volumes

| Path | Purpose |
|------|---------|
| `{{ composition_config }}/collections` | CalDAV collections (reminder lists) |
| `{{ composition_config }}/users` | htpasswd auth file, generated by the `Create htpasswd file` task from vault-stored credentials |

## DNS

Registers subdomain: `radicale`
