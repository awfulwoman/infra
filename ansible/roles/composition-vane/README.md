# Vane

Privacy-focused AI answering engine that combines internet knowledge with local LLMs via Ollama.

* [Homepage](https://github.com/ItzCrazyKns/Vane)

## Configuration

Ollama and model selection are configured through the web UI on first launch.

The default Ollama API URL is `http://192.168.1.150:11434` and can be overridden via the `vane_ollama_api_url` variable.
