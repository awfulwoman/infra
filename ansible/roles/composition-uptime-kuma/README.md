# Uptime Kuma

Uptime Kuma is a self-hosted monitoring tool that allows you to monitor uptime for HTTP(s) / TCP / HTTP(s) Keyword / Ping / DNS Record / Push / Steam Game Server / Docker Containers.

- **Image**: louislam/uptime-kuma:1
- **Port**: 3001
- **Data**: Stored in /app/data

## Features

- Monitoring uptime for HTTP(s) / TCP / HTTP(s) Keyword / Ping / DNS Record / Push / Steam Game Server
- Fancy, Reactive, Fast UI/UX
- Notifications via Telegram, Discord, Gotify, Slack, Pushover, Email (SMTP), and 90+ notification services
- 20 second intervals
- Multi Languages
- Multiple Status Pages
- Map Status Page to Domain
- Ping Chart
- Certificate Info
